# Remote-Controlled-RC-Smart-Car
This project features an intelligent mobile robot system controlled via a smartphone.
Developed using MicroPython within the Thonny IDE, the system implements a robust Bluetooth serial communication protocol. By transmitting commands from a mobile app, the car processes real-time signals to drive motor modules, enabling precise maneuvers including forward, reverse, and steering.
